# DevOps Lab

Mon parcours d'apprentissage DevOps, de zéro à opérationnel.

## Ce que contient ce repo

- **diagnostic.sh** — Script bash de diagnostic serveur (disque, mémoire, alertes)
- **app-complete/** — App Flask + Redis avec Docker Compose
- **terraform-lab/** — Infrastructure as Code avec Terraform
- **.github/workflows/** — Pipeline CI/CD avec GitHub Actions

## Compétences pratiquées

- Linux & Bash
- Git
- Docker & Docker Compose
- CI/CD (GitHub Actions)
- Terraform
- AWS (EC2)
